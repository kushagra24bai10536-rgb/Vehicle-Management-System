# Vehicle Management System 🚗

A high-performance management application designed to track vehicle lifecycles, maintenance schedules, and fleet utilization. This system provides a centralized dashboard for monitoring fuel efficiency, service records, and driver assignments.

## 🌟 Key Features
* **Inventory Tracking:** Detailed records of vehicle make, model, year, and registration status.
* **Maintenance Scheduler:** Automated logs for oil changes, tire rotations, and insurance renewals.
* **Fuel Management:** Track fuel consumption and cost-per-mile analytics.
* **Driver Assignment:** Link vehicles to specific drivers and monitor trip logs.
* **Status Monitoring:** Real-time visibility (In Service, Under Repair, or Available).
